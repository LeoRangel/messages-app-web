
- Criar projeto
yarn create vite messages-app-web --template react-ts

- Instalar dependências
yarn

- Instalar pré processador de css
yarn add sass

- Instalar lib de icons
yarn add react-icons

- Instalar lib para acessar api's
yarn add axios

- Instalar cliente do web socket para ouvir mensagens em tempo real
yarn add socket.io-client
